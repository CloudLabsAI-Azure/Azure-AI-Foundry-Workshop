# Exercise 1: Set Up the Environment

This lab focuses on setting up the foundational environment for developing AI applications using the Azure AI Foundry SDK. Participants will establish their development workspace, configure necessary connections, and authenticate credentials to ensure seamless integration with Azure AI services.

### Lab Overview

In this lab, you will set up the AI Foundry environment by cloning the necessary repositories and configuring a Python-based development environment. You'll deploy essential connections and verify access to models, ensuring smooth execution of AI workflows. By completing this setup, you will be prepared to leverage Azure AI Foundry for building advanced AI solutions.
